# Insurance Policy Q&A Chatbot

An intelligent chatbot application that enables users to upload insurance policy documents and ask questions about their coverage, using Retrieval-Augmented Generation (RAG) and Claude 3.8 Sonnet.

## Overview

This project demonstrates a practical implementation of modern AI/ML tools for document analysis and question-answering:

- **LLM**: Claude 3.8 Sonnet via Anthropic API
- **Framework**: LangChain for orchestration
- **Vector Store**: ChromaDB for document embeddings
- **Containerization**: Docker for deployment
